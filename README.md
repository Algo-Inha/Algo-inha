# 브렌치 규칙

`feature/(내이름)-(문제를 식별할 수단)`
<br/> ex) `feature/adultlee-baekjoon-21751`
<br/> 문제를 식별할 수단은 무엇이든 상관 없습니다.

- solve: ~

- write: ~
